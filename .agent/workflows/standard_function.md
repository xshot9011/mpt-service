description: Implement a function following the standard 4-step process (Validate, Process, Store, Return)
---

When asking me to implement a function using this workflow, I will strictly follow these steps:

1.  **Validate Input**: Ensure all arguments meet requirements (types, ranges, existence) before proceeding. Return early or raise errors if invalid.
2.  **Process Logic**: Execute the core business logic or transformation.
3.  **Store**: Persist the result to a database, state, or variable if applicable.
4.  **Return Output**: Return the final result in the expected format.

**Usage:**
- "Implement the [function_name] function following the standard workflow."
